## 5-Minute Quick Start

### Step 1: Enter API Key (Sidebar)
Get from IT team ‚Üí Enter in "OpenAI API Key" field ‚Üí See green checkmark

### Step 2: Enter Product Info (Left Panel)
- Product name: e.g., "TEREA Clear Regular"
- Flavor: e.g., "Refined, Clear, Subtle Lemon"
- Price: e.g., "¬•580" (optional)
- Launch date: e.g., "October 27, 2025" (optional)

### Step 3: Upload Product Photo
Click "Browse files" ‚Üí Select product image ‚Üí See preview

### Step 4: Choose Design Tone
Select ONE:
- **Premium & Sophisticated** ‚Üê Most common (elegant, gold lighting, multi-tier)
- **Striking & Powerful** ‚Üê Bold statement (dramatic, angular)
- **Bold & Aggressive** ‚Üê High energy (neon, sharp angles)
- **Minimal & Clean** ‚Üê Understated (simple, monochrome)
- **Vibrant & Energetic** ‚Üê Colorful (bright, playful)

### Step 5: Pick Your Colors
Click color boxes to choose:
- **Primary**: Main panel color (default: white)
- **Accent**: Lighting color (default: gold)
- **Base**: Platform color (default: black)

### Step 6: Generate
Click "Generate Floorbin Design" button ‚Üí Wait 30-60 seconds ‚Üí Download result

## Typical Workflow

**Exploration Phase** (30 min)
- Generate 3 different tones
- Same product, different aesthetics
- Choose preferred direction

**Refinement Phase** (20 min)
- Stick with chosen tone
- Try 2-3 color variations
- Test design element combinations

**Selection Phase** (10 min)
- Generate final 2-3 candidates
- Download all
- Present to stakeholders

**Total: ~1 hour for complete concept exploration**
(vs. Days/weeks traditional process)

## Troubleshooting 3-Step Fix

**Problem: Generation Failed**
1. Check API key entered correctly
2. Verify internet connection
3. Try again or switch AI model

**Problem: Design Looks Wrong**
1. Verify reference image uploaded
2. Check design tone selection
3. Review color picker settings

**Problem: Product Doesn't Match**
1. Re-upload reference image
2. Use higher quality photo
3. Regenerate
